## Instalación

1. Clonar el repositorio:

    ```bash
    git clone <reponame>
    ```

2. Instalar las dependencias:

    ```bash
    npm install
    ```

3. Iniciar el servidor:

    ```bash
    npm run dev
    ```

## Stack Tecnológico

**Cliente:** React, Vite, React Router, styled-components, useContext, useState, React Icons.
